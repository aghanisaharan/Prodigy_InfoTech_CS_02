# Pixel Manipulation for Image Encryption 

This repository contains my Python program for Task 02 of the Cyber Security Internship at **Prodigy InfoTech**. 

The objective of this project is to develop a simple image encryption tool using pixel manipulation. 

## Features
* **Encryption & Decryption:** Capable of both encrypting and decrypting images using a user-provided numeric key.
* **Pixel Swapping:** Modifies images by securely shuffling RGB pixel values based on a seeded random generation.
* **Format Preservation:** Saves the processed images with dynamic filenames using the `os` module.

## Requirements
* Python 3.x
* Pillow (PIL) library (`pip install Pillow`)
